この前投稿したRatatanの記事。
[MacにてRatatanのアーリーアクセス版ができた！](/2025/09/23//ratatan-on-mac)



まあ、Ratatanは上記でインストールして子どもと楽しめているのだが、
今まで買った他のゲームで動くのは、FactorioとInscryptionぐらいで、
Civilizationとかは動かないなーって思っていたのだけど、
よくよく考えたら、Civ5,6はMac版が出ているんだよね。

ということで、Mac版のSteamをインストールして動くかどうか試してみる


まずはbrewでsteamいれる。 もうcaskいらないんだっけ？まあいいや。つけちゃった。

```bash
$ brew install --cask steam

```

たぶんMac のSiliconチップだから、Rosetta 2が必要。
すでに入っているから別に何もしなかった。

インストール終わったら起動してログイン。



で、Civ 5, 6をダウンロードしてくる。
Civ 5は日本語版はWindowsしかないが、Civ 6はMac版でもついているらしい。


まず容量が軽いCiv 5から動かしてみると、起動はするんだけど、Loading画面で落ちちゃう。

で、調べてみると下記が見つかった。

[Civ 5 on Mac OS - Workaround for the Steam Client Bug : r/civ5](https://www.reddit.com/r/civ5/comments/j9jbii/civ_5_on_mac_os_workaround_for_the_steam_client/)


steam_appid.txtファイルが必要とのこと。

ということで下記でCiv5アプリの下に、steam_appid.txtを追加する。中身は8930 (Civ5のsteam id)


```bash
$ cd ~/Library/Application\ Support/Steam/steamapps/common/Sid\ Meier\'s\ Civilization\ V/Civilization\ V.app/Contents/MacOS

$ echo 8930 > steam_appid.txt

```

なんでこれで動くのかよくわからないけど、steam再起動したら、無事にCiv 5起動した。




なお、Civ 6 は、特に何もしなくても普通に動いた。



まじかー。Civ 7が発売している今日このごろだけど、Civ 6で数百時間がまた溶けてしまうなー。
