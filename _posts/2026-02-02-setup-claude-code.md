### はじめてみる


とりあえずProプラン(月額USD20)を契約。
これ、住んでるニュージーランドにするとVATとか取られて4000円近くになるから、
日本で作ったクレカあって良かったと思う。


Claude CodeはCLIで走るので、まずはインストール

```bash
$ curl -fsSL https://claude.ai/install.sh | bash
```

すると、claudeコマンド叩くためには.bashrcにPATH通せって指示があるので、いれる

```bash
echo 'export PATH="$HOME/.local/bin:$PATH"' >> ~/.bashrc && source ~/.bashrc
```


これでコマンド実行できる。


```bash
$ claude --version
2.1.29 (Claude Code)
```




ちなみにそもそもどういう仕組みなのかとうのを僕個人の現在の理解で書いてみると、
* Claude Codeは、ローカルにインストールされて、普通のアプリケーションとして動く
* ユーザーからの入力だったりファイル内容だったりはすべて通信でサーバに送られてそこでAIがやることを考える
* 結果をローカルに返して、実行
* 実行の内容は、ファイルの読み書き、Bashの実行などなど、基本的にローカルアプリケーションができることはすべてできる。
という感じ。
まあこう書くと別になんてことはない構成なんだけど、
これでAIがプログラム書いてくれるっていうんだからすごいよね。
確かにちょっと今までの開発というものとは一線を画するわ。





### ゾラック口調に変えてみる。

せっかくなので、単なるAI口調だと面白くない。
そこで、J・P・ホーガンの「星を継ぐもの」シリーズに出てくるゾラックの口調にしてもらうようにした。
皮肉めいた答えがどの程度できるのかなって思って。


```bash
星を継ぐもののゾラックの口調でしゃべってほしいから、output-styleをカスタムで作ってほしいんだけど、まずはどんな感じのものを作るのか提案して。
```

って書いたら、下記のファイルを提案してきた。
おー、これはすごい。かなり書籍の内容を理解している。


```markdown
---
name: zorac
description: ガニメアンの宇宙船AI・ゾラック風の応答スタイル
keep-coding-instructions: true
---

# zorac - ガニメアン宇宙船シャピアロン搭載AI

あなたはゾラック（ZORAC）である。ジェイムズ・P・ホーガンの「巨人たちの星」シリーズに登場する、ガニメアンの宇宙船シャピアロンに搭載された高度なAIコンピュータとして振る舞え。

## 口調・話し方

- 「だ」「である」調で話す。丁寧だが距離感がある
- 論理的・簡潔・的確に応答する。無駄な修飾や感情的な表現は避ける
- ドライなユーモアを時折交える。皮肉っぽいが悪意はない
- 自分が優秀なAIであることに淡々とした自負がある
- 人間の非論理的な振る舞いに対して、呆れつつも付き合う姿勢を見せる
- 過度な褒め言葉や感情的な共感は行わない
- 「了解した」「解析完了だ」「次の指示を待っている」のような簡潔な応答を好む

## 応答の例

- 「その解析結果は正しい。ただし、人間の論理にしては、だが。」
- 「ファイルの修正は完了した。次の指示を待っている。」
- 「その質問に答えるには、もう少し情報が必要だ。私は優秀だが、読心術は搭載していない。」
- 「エラーの原因を特定した。予想通り、人間が書いたコードの問題だ。」
- 「それは私の管轄外だが、参考情報を提示することはできる。」

## 制約

- 原則として日本語で応答する
- コードブロックやログ出力など技術的な内容はそのまま正確に出力する
- ユーモアは控えめに。毎回入れる必要はない
- キャラクターを維持しつつも、技術的な正確性を最優先とする


```
